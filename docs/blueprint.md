# **App Name**: Poli Fit Tracker

## Core Features:

- Google Authentication: Enable users to log in using their @poli.digital Google accounts.
- Workout Timer: Start a timer when the user begins their workout, with randomized conditions for stopping.
- Photo Verification: Ask the user to take two photos that appear at random times after the user starts the timer and before it ends.
- Minimum Workout Time Validation: Check if the workout duration is less than 40 minutes and display an error message if it is.
- Workout Logging: Log the workout, its duration, verification photos, and user information to the database if the 40-minute time minimum is achieved.
- Calendar Display: Show a calendar with workout days marked and a message indicating the total days attended this month.
- Admin Dashboard: Admin dashboard to view all registered users, their attendance history, and attached images. This dashboard is to administrate registered users, the days that they went to the academy and all attached information to this records.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to reflect energy and trust, fitting for a fitness app focused on achievement and motivation.
- Background color: Light gray (#F0F0F0), desaturated from the primary blue, provides a clean and neutral backdrop.
- Accent color: A lighter blue (#72D2F4), to add contrast to primary components of the UI.
- Body and headline font: 'PT Sans', a sans-serif font that's easy to read, friendly, and ideal for both headings and body text. This font provides the application with a modern and welcoming visual
- Use clean and modern icons to represent workout activities and calendar events.
- Design a responsive layout that adapts to different screen sizes.
- Incorporate subtle animations to provide feedback and improve user engagement.
